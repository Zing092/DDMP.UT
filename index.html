ขั้นตอนการตั้งค่า Firebase
  1. สร้าง Firebase Project ที่ console.firebase.google.com
  2. เปิดใช้งาน Authentication → Email/Password
  3. สร้าง Firestore Database
  4. สร้าง Storage
  5. คัดลอก Firebase Config ใส่ในโค้ดส่วน firebaseConfig ในไฟล์นี้
  6. แก้ไข Firestore Rules เพื่อรองรับการใช้งาน
  7. สร้างแอดมินแรกผ่าน Firebase Console (Authentication → Add user)
  8. เพิ่มข้อมูล users collection ด้วย: {empid, name, position, role:"admin"}
